## nextjs-typescript-redux-saga

This is a Nextjs shop store web application .
It has ben implemented by :
- nextjs
- redux
- redux-saga
- axios
- sass
- typescript

## [Demo](https://silly-darwin-9bb841.netlify.app/)


## How to use

Install it and run:

```bash
npm install
npm run dev
# or
yarn
yarn dev
```
